# NajaSci

A starter repo to handle statistic data for academic

## Architecture

![](architecture.png)


## Local

##### Install

```
pip install --no-cache-dir --root-user-action=ignore -r requirements.txt
```

##### Run

```
python index.py
```

## Docker

##### Install

```
./script/install.sh
```

##### Run

```
./script/run.sh
```
